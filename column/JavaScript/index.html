浅拷贝和深拷贝 ​

浅拷贝 ​

一般是指把对象的第一层拷贝到一个新对象上去 实现浅拷贝的几种方式：

  • slice

  • concat

  • 扩展运算符 [...arr]

  • 扩展运算符

  • Object.assign 对象浅拷贝

js
let a = { count: 1, deep: { count: 2 } }
let a1 = { count: 2, deep: { count: 4 } }
let b = Object.assign({}, a, a1)
let c = JSON.parse(JSON.stringify(a))
a1.deep.count = 3
console.log(b)
console.log(c)

数组浅拷贝：

js
let arr = [1, { a: 2 }]

let arr2 = arr.slice(0)
arr2 = arr.concat()
arr2 = [...arr]

arr2[1].a = 3
console.log(arr) // [1, { a: 3 }]

事件冒泡和事件捕获 ​

  1. 事件冒泡是微软提出的，后网景公司提出事件捕获，都是为了解决页面中事件流（事件发生顺序）的问题
  2. 事件代理：将子元素的事件绑定在父元素上，不用逐个绑定；好处是新增子元素时，无需再对其进行绑定
  3. e.target 和 this 区别：前者是触发事件的元素，后者是绑定事件的元素
  4. 阻止事件冒泡的方式：
  • e.stopPropagation
  • return false（阻止冒泡外，还会阻止事件本身）
  • e.target === e.currentTarget（让触发事件的元素等于绑定事件的元素）
  1. addEventListener 事件绑定第三个属性是 useCapture
