## [[prototype]]

对象有一个隐藏的内部属性 [[prototype]]，要么为 null，要么就是对另一个对象的引用，这个对象被称为“原型”

当从对象中获取一个缺失的属性时，js 会自动从原型中获取该属性，这被称为“原型继承”

```javascript
let animal = {
  eats: true,
};
let rabbit = {
  jumps: true,
};
rabbit.__proto__ = animal; // 设置 rabbit.[[prototype]] = animal

console.log(rabbit.eats); // true
```
