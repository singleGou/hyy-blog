一、先修条件（7 天快速过）

1. 掌握一门主力语言（Python / C++ / Java 均可）
2. 会用 Git 提交代码、会用 IDE 调试
3. 了解大 O 记法的基本含义（时间/空间复杂度）

------

三、免费/高质量在线资源（按类型）

1. 图文 + 动画
   - Hello 算法（多语言示例 + 动画图解）[https://www.hello-algo.com](https://www.hello-algo.com/)
   - VisualGo（交互式算法动画）https://visualgo.net/zh
2. 视频课程
   - 清华大学《数据结构与算法》Coursera 系列（4 门课，中英字幕）
   - 科罗拉多大学《数据结构基础》Coursera（含树、图、动规）
   - B 站“小甲鱼”Python 数据结构与算法（零基础友好）
3. 刷题/竞赛
   - LeetCode（面试向）
   - Codeforces / AtCoder（竞赛向）
   - 牛客网《剑指 Offer》高频题单 
4. 电子书
   - 《大话数据结构》《算法图解》适合入门
   - 《算法导论》《数据结构与算法分析》当工具书查

四、学习建议（总结自多篇经验）

- 先“看懂”→ 再“默写”→ 最后“举一反三”，每道经典题至少重复 3 遍。
- 每学完一个结构，立刻在 LeetCode/牛客找 3-5 题巩固；不会就先看题解，再闭卷写。
- 把代码推到 GitHub，用 README 记录思路与复杂度，方便复习与展示。
- 参加周赛（CF/AtCoder/LeetCode Contest）检验实战速度和心态。
- 坚持 3-6 个月，每周 8-10 题，面试/竞赛都能游刃有余
